# AI Resume Analyzer

An AI-powered Resume Analyzer built with React, React Router v7, TypeScript, Tailwind CSS, and Puter.js.

## Features

* Resume upload and management
* ATS score analysis
* AI-powered resume feedback
* Resume-job matching
* Responsive modern UI
* Secure authentication

## Tech Stack

* React
* React Router v7
* TypeScript
* Tailwind CSS
* Puter.js
* Zustand
* Vite

## Installation

```bash
npm install
```

## Run the Project

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
